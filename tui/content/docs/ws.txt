# VLESS + WebSocket + CDN

Route traffic through Cloudflare CDN.
Hides your server IP completely.

## Why WebSocket + CDN

- Server IP hidden behind Cloudflare
- Survives IP-based blocking
- Uses Cloudflare's global network
- Fallback when direct connections blocked

## When to Use

- Direct connections to server are blocked
- Need to hide server IP
- Have a domain on Cloudflare
- As backup when REALITY stops working

## How It Works

Traffic goes through Cloudflare's CDN via WebSocket.
Censors only see connections to Cloudflare IPs.

  Client --> Cloudflare CDN --> Your Server --> Internet
    |             |                 |
    |  HTTPS/WS   |  Re-encrypted   |
    |             |  to origin      |

## Setup Notes

- Requires a domain name on Cloudflare
- Cloudflare SSL mode must be "Flexible"
- Port 80 must be open on server
- Cloudflare proxy must be enabled (orange cloud)
